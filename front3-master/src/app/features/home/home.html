Sistema de Monitoreo Ambiental Inteligente

Greenhouse IoT

Sistema basado en sensores IoT que permite monitorear en tiempo real variables ambientales como temperatura, humedad del aire y humedad del suelo dentro de un invernadero.

📡 Descripción del sistema

Este proyecto integra sensores físicos conectados a un sistema IoT que captura datos ambientales de forma automática y los envía a una plataforma web para su visualización y análisis.

⚙️ Funcionamiento

Los sensores registran continuamente variables ambientales, las cuales son procesadas en el backend y visualizadas en tiempo real en la interfaz del sistema.

📡 Captura de datos

Sensores IoT registran temperatura, humedad del aire y humedad del suelo en tiempo real.

📊 Visualización

Los datos se muestran mediante gráficos interactivos para facilitar el análisis del entorno.

🔌 Integración IoT

Comunicación entre hardware y backend para transmisión automática de datos.

🌿 Aplicaciones

Agricultura inteligente, monitoreo de invernaderos y proyectos de ingeniería IoT.

📈 Impacto

Mejora la eficiencia del cultivo mediante monitoreo continuo y toma de decisiones basada en datos.
